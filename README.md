Este é um bot que administra a fila do Ping Pong na UFC de Quixadá utilizando Node.js!

## ⚡ Setup

### 1. 📦 Instale os pacotes

```shell
npm install
```

### 2. 📃 Crie um arquivo .env

```env
TOKEN = TOKEN DO BOT
MODS = "usuário1, usuário2, usuário3"
````

### 2. ⚙️ Rode o servidor

```shell
npm start
```
